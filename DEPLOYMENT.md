# Render এ ডিপ্লয়মেন্ট গাইড

এই গাইডটি আপনার ডায়াবেটিস প্রেডিকশন অ্যাপ্লিকেশন Render.com এ ডিপ্লয় করার জন্য ধাপে ধাপে নির্দেশনা প্রদান করে।

## প্রয়োজনীয়তা

1. **GitHub অ্যাকাউন্ট** - যদি না থাকে তাহলে [github.com](https://github.com) এ তৈরি করুন
2. **Render অ্যাকাউন্ট** - [render.com](https://render.com) এ সাইন আপ করুন (GitHub দিয়ে করতে পারেন)
3. **আপনার প্রজেক্ট** - সমস্ত ফাইল সহ GitHub রিপোজিটরিতে

## ধাপ 1: GitHub এ রিপোজিটরি সেটআপ করুন

### ১.১ নতুন রিপোজিটরি তৈরি করুন

1. GitHub এ লগইন করুন
2. **+** আইকনে ক্লিক করুন এবং **New repository** নির্বাচন করুন
3. রিপোজিটরি নাম দিন (যেমন: `diabetes-prediction`)
4. **Public** নির্বাচন করুন
5. **Create repository** ক্লিক করুন

### ১.২ আপনার ফাইল পুশ করুন

স্থানীয় টার্মিনালে নিম্নলিখিত কমান্ড চালান:

```bash
# রিপোজিটরি ইনিশিয়ালাইজ করুন (যদি এখনো না করা হয়)
git init

# সমস্ত ফাইল যোগ করুন
git add .

# প্রথম কমিট করুন
git commit -m "Initial commit: Diabetes prediction ML application"

# রিমোট অরিজিন যোগ করুন (HTTPS এ পরিবর্তন করুন)
git remote add origin https://github.com/YOUR_USERNAME/diabetes-prediction.git

# main ব্র্যাঞ্চে পুশ করুন
git push -u origin main
```

**নোট:** `YOUR_USERNAME` আপনার GitHub ইউজারনাম দিয়ে প্রতিস্থাপন করুন।

## ধাপ 2: Render এ ডিপ্লয় করুন

### ২.১ নতুন ওয়েব সার্ভিস তৈরি করুন

1. [Render Dashboard](https://dashboard.render.com) এ যান
2. **New +** বাটনে ক্লিক করুন
3. **Web Service** নির্বাচন করুন

### ২.২ GitHub সংযোগ করুন

1. **Connect your GitHub account** বিকল্পে ক্লিক করুন
2. GitHub এ অনুমোদন দিন
3. আপনার `diabetes-prediction` রিপোজিটরি খুঁজে বের করুন এবং **Connect** করুন

### ২.३ সার্ভিস কনফিগার করুন

নিম্নলিখিত তথ্য পূরণ করুন:

| সেটিং | মূল্য |
|--------|-------|
| **Name** | `diabetes-prediction` (আপনার পছন্দের নাম) |
| **Environment** | `Python 3` |
| **Region** | `Singapore` (বা আপনার কাছাকাছি অঞ্চল) |
| **Branch** | `main` |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `gunicorn app:app` |

### २.४ পরিবেশ ভেরিয়েবল সেট করুন (ঐচ্ছিক)

**Environment** ট্যাবে যান এবং যদি প্রয়োজন হয় এখানে যোগ করুন:

```
FLASK_ENV=production
PYTHONUNBUFFERED=1
```

### २.५ ইনস্ট্যান্স টাইপ নির্বাচন করুন

- **Starter** ($7/মাস) - ছোট প্রজেক্টের জন্য যথেষ্ট
- **Standard** ($12/মাস) - ভাল পারফরম্যান্সের জন্য

আপনার প্রয়োজন অনুযায়ী নির্বাচন করুন।

### २.६ সার্ভিস তৈরি করুন

**Create Web Service** বাটনে ক্লিক করুন এবং অপেক্ষা করুন।

## ধাপ 3: ডিপ্লয়মেন্ট মনিটর করুন

1. Render Dashboard এ আপনার সার্ভিস নাম ক্লিক করুন
2. **Logs** ট্যাবে দেখুন যে সবকিছু সঠিকভাবে তৈরি হচ্ছে কিনা
3. যখন দেখবেন **"Server running on http://localhost:10000"** তখন বুঝবেন সফল হয়েছে

## ধাপ 4: লাইভ সাইট অ্যাক্সেস করুন

ডিপ্লয়মেন্ট সম্পন্ন হলে, Render আপনাকে একটি **public URL** দেবে যেমন:

```
https://diabetes-prediction.onrender.com
```

এই লিংকটি ব্রাউজারে খুলুন এবং আপনার অ্যাপ্লিকেশন চেষ্টা করুন!

## সমস্যা সমাধান

### সমস্যা 1: বিল্ড ব্যর্থ হচ্ছে

**সমাধান:**
- Logs চেক করুন কী ত্রুটি আছে
- নিশ্চিত করুন `requirements.txt` সঠিক এবং আপডেট
- `Procfile` এ কোনো টাইপো নেই কিনা যাচাই করুন

### সমস্যা 2: অ্যাপ ক্রাশ হচ্ছে

**সমাধান:**
- Logs দেখুন বিস্তারিত ত্রুটির জন্য
- নিশ্চিত করুন সমস্ত ডিপেন্ডেন্সি `requirements.txt` তে আছে
- পিকল মডেল ফাইল রিপোজিটরিতে আছে কিনা চেক করুন

### সমস্যা 3: মডেল লোড হচ্ছে না

**সমাধান:**
```bash
# পিকল মডেল ফাইল ট্র্যাক করা আছে কিনা চেক করুন
git lfs install
git lfs track "*.pkl"
git add .gitattributes pickel_model.pkl
git commit -m "Add model file"
git push
```

### সমস্যা 4: পূর্বাভাস ত্রুটি দিচ্ছে

**সমাধান:**
- ব্রাউজার ডেভেলপার টুলস (F12) এ Console খুলুন
- নেটওয়ার্ক রেসপন্স দেখুন
- Render Logs চেক করুন সার্ভার-সাইড ত্রুটির জন্য

## কন্টিনিউয়াস ডিপ্লয়মেন্ট

Render স্বয়ংক্রিয় ডিপ্লয়মেন্ট সমর্থন করে। যখন আপনি GitHub এ পুশ করবেন:

```bash
git add .
git commit -m "Update features"
git push origin main
```

Render স্বয়ংক্রিয়ভাবে নতুন সংস্করণ তৈরি এবং ডিপ্লয় করবে!

## কাস্টম ডোমেইন (ঐচ্ছিক)

Render থেকে একটি নিজস্ব ডোমেইন সংযুক্ত করতে:

1. Dashboard এ সার্ভিস খুলুন
2. **Settings** ট্যাবে যান
3. **Custom Domain** ত্যাগে আপনার ডোমেইন যোগ করুন
4. DNS সেটিংস অনুসরণ করুন
