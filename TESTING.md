# স্থানীয় পরীক্ষা গাইড

আপনার ডায়াবেটিস প্রেডিকশন অ্যাপ্লিকেশন স্থানীয়ভাবে চালানো এবং পরীক্ষা করার জন্য এই গাইড অনুসরণ করুন।

## পরিবেশ সেটআপ

### ১. ভার্চুয়াল এনভায়রনমেন্ট তৈরি এবং সক্রিয় করুন

**Windows:**
```bash
# প্রজেক্ট ফোল্ডারে যান
cd d:\ML\ model

# ভার্চুয়াল এনভায়রনমেন্ট তৈরি করুন (যদি না করা হয়)
python -m venv myenv

# সক্রিয় করুন
myenv\Scripts\activate
```

**macOS/Linux:**
```bash
cd ~/diabetes-prediction
python3 -m venv venv
source venv/bin/activate
```

## ফিচার পরীক্ষা করুন

### ১. ফর্ম যাচাইকরণ পরীক্ষা করুন

সমস্ত ক্ষেত্র সহ ফর্ম পূরণ করুন:

| ক্ষেত্র | নমুনা মূল্য | বিবরণ |
|---------|------------|-------|
| গর্ভধারণের সংখ্যা | 6 | পূর্ববর্তী গর্ভধারণের সংখ্যা |
| গ্লুকোজ | 148 | রক্তে গ্লুকোজ স্তর |
| ব্লাড প্রেশার | 72 | রক্তচাপ |
| ত্বকের পুরুত্ব | 35 | ত্রিসেপস ত্বকের ফোল্ড |
| ইনসুলিন | 0 | ২ ঘণ্টা সিরাম ইনসুলিন |
| BMI | 33.6 | বডি মাস ইন্ডেক্স |
| ডায়াবেটিস পারিবারিক ইতিহাস | 0.627 | পারিবারিক ইতিহাস ফাংশন |
| বয়স | 50 | বছরে বয়স |

**পূর্বাভাস দিন** বাটনে ক্লিক করুন।

### २. পূর্বাভাস ফলাফল যাচাই করুন

ফলাফল হওয়া উচিত:
- ✅ পূর্বাভাস লেবেল (ডায়াবেটিস আছে/নেই)
- ✅ আত্মবিশ্বাস স্তর (শতাংশ)
- ✅ উপযুক্ত রঙ কোড (সবুজ/লাল)

### ३. নতুন পরীক্ষা করুন বাটন পরীক্ষা করুন

**নতুন পরীক্ষা করুন** বাটনে ক্লিক করুন এবং ফর্ম রিসেট হওয়া উচিত।

## API এন্ডপয়েন্ট পরীক্ষা করুন

### cURL দিয়ে প্রত্যক্ষ API পরীক্ষা

```bash
curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "pregnancies": 6,
    "glucose": 148,
    "blood_pressure": 72,
    "skin_thickness": 35,
    "insulin": 0,
    "bmi": 33.6,
    "diabetes_pedigree": 0.627,
    "age": 50
  }'
```

**প্রত্যাশিত প্রতিক্রিয়া:**
```json
{
  "prediction": 1,
  "prediction_text": "ডায়াবেটিস আছে",
  "confidence": 75.5
}
```

### পাইথন দিয়ে API পরীক্ষা

```python
import requests
import json

# এন্ডপয়েন্ট
url = "http://localhost:5000/predict"

# টেস্ট ডেটা
data = {
    "pregnancies": 1,
    "glucose": 85,
    "blood_pressure": 66,
    "skin_thickness": 29,
    "insulin": 0,
    "bmi": 26.6,
    "diabetes_pedigree": 0.351,
    "age": 31
}

# পোস্ট রিকোয়েস্ট
response = requests.post(url, json=data)

# ফলাফল প্রিন্ট করুন
print("স্ট্যাটাস কোড:", response.status_code)
print("প্রতিক্রিয়া:", json.dumps(response.json(), ensure_ascii=False, indent=2))
```

## ব্রাউজার ডেভেলপার টুলস পরীক্ষা

## পারফরম্যান্স পরীক্ষা

### ১. প্রতিক্রিয়া সময় চেক করুন

```bash
# Windows
time curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{"pregnancies": 6, "glucose": 148, ...}'
```

প্রতিক্রিয়া সময় **< 500ms** হওয়া উচিত।

### २. একাধিক অনুরোধ পাঠান

```python
import time
import requests

url = "http://localhost:5000/predict"
data = {"pregnancies": 6, "glucose": 148, ...}

# 10টি অনুরোধ পাঠান
for i in range(10):
    start = time.time()
    response = requests.post(url, json=data)
    duration = time.time() - start
    print(f"অনুরোধ {i+1}: {duration:.3f} সেকেন্ড")
```
