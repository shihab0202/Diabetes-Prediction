# ডায়াবেটিস প্রেডিকশন সিস্টেম

একটি মেশিন লার্নিং ভিত্তিক ওয়েবসাইট যা ডায়াবেটিস ঝুঁকি পূর্বাভাস দিতে সাহায্য করে।

## বৈশিষ্ট্য

- 🎯 **নির্ভুল পূর্বাভাস**: উন্নত মেশিন লার্নিং মডেল ব্যবহার
- 🌐 **ব্যবহারকারী-বান্ধব ইন্টারফেস**: সহজ এবং দ্রুত ডেটা প্রবেশ
- 📱 **প্রতিক্রিয়াশীল ডিজাইন**: সকল ডিভাইসে কাজ করে
- 🔒 **নিরাপদ**: ডেটা সরাসরি সার্ভারে প্রক্রিয়া করা হয়
- 🚀 **দ্রুত পূর্বাভাস**: তাৎক্ষণিক ফলাফল

## প্রযুক্তি স্ট্যাক

- **ব্যাকএন্ড**: Flask (Python)
- **ফ্রন্টএন্ড**: HTML5, CSS3, Vanilla JavaScript
- **মেশিন লার্নিং**: scikit-learn
- **ডিপ্লয়মেন্ট**: Render.com

## স্থানীয় সেটআপ

### প্রয়োজনীয়তা
- Python 3.8 বা তার উপরে
- pip (Python প্যাকেজ ম্যানেজার)

### ইনস্টলেশন

1. **রিপোজিটরি ক্লোন করুন**
```bash
git clone <your-repository-url>
cd diabetes-prediction
```

2. **ভার্চুয়াল পরিবেশ তৈরি করুন**
```bash
python -m venv venv
```

3. **ভার্চুয়াল পরিবেশ সক্রিয় করুন**

**Windows এ:**
```bash
venv\Scripts\activate
```

**macOS/Linux এ:**
```bash
source venv/bin/activate
```

4. **ডিপেন্ডেন্সি ইনস্টল করুন**
```bash
pip install -r requirements.txt
```

5. **অ্যাপ চালান**
```bash
python app.py
```

6. **ব্রাউজারে খুলুন**
```
http://localhost:5000
```

## Render এ ডিপ্লয়মেন্ট

### ধাপ 1: GitHub এ পুশ করুন
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

### ধাপ 2: Render এ ডিপ্লয় করুন

1. [Render.com](https://render.com) এ যান এবং লগইন করুন
2. **New +** বাটনে ক্লিক করুন
3. **Web Service** নির্বাচন করুন
4. আপনার GitHub রিপোজিটরি সংযুক্ত করুন
5. নিম্নলিখিত সেটিংস কনফিগার করুন:
   - **Name**: diabetes-prediction (বা আপনার পছন্দের নাম)
   - **Environment**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app`
6. **Create Web Service** ক্লিক করুন

### ধাপ 3: যাচাই করুন
ডিপ্লয়মেন্ট সম্পন্ন হলে, আপনার লাইভ ওয়েবসাইট অ্যাক্সেস করুন।

## API এন্ডপয়েন্ট

### `POST /predict`
ডায়াবেটিস ঝুঁকি পূর্বাভাস দেয়।

**Request:**
```json
{
    "pregnancies": 6,
    "glucose": 148,
    "blood_pressure": 72,
    "skin_thickness": 35,
    "insulin": 0,
    "bmi": 33.6,
    "diabetes_pedigree": 0.627,
    "age": 50
}
```

**Response:**
```json
{
    "prediction": 1,
    "prediction_text": "ডায়াবেটিস আছে",
    "confidence": 75.5
}
```
